# GitHub Copilot Rules for this Workspace

## Korean Comment Rules

- When writing Korean comments, always use '번째' instead of '째' or '쨰' (e.g., '0번째', '1번째').
- Avoid typos caused by holding the Shift key too long (like '째' becoming '쨰').
- Ensure all ordinal numbers follow the standard Korean spelling: [Number]번째.
